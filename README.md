# 🏥 MediGuide – Intelligent Prescription Analyzer

MediGuide is a web-based application that helps users understand medical prescriptions by extracting medicines using OCR and providing useful insights like dosage, diet recommendations, and pharmacy purchase options.

---

## 🚀 Features

- 🔍 AI Prescription Analysis  
  Extracts medicine names and dosage from prescription images  

- 💊 Medicine Details  
  Displays medicine name with correct dosage  

- 🥗 Diet Recommendations  
  Suggests diet based on medicines  

- ⚠️ Safety Suggestions  
  Provides basic usage instructions  

- 🌐 Multilingual Support  
  Supports English, Hindi, and Telugu  

- 🛒 Pharmacy Finder  
  Redirects users to purchase medicines online  

- 📄 Prescription Summary  
  Displays raw OCR extracted text  

---

## ⚙️ How It Works

1. Upload prescription image  
2. OCR extracts raw text  
3. Medicines and dosage are identified  
4. Data is matched with dataset  
5. Displays:
   - Extracted medicines  
   - Diet recommendations  
   - Suggestions  
6. User can purchase medicines via pharmacy links
